# kinoapi-frontend

Webapp to control cinema tech:
- start/pause the projector
- control the volume
- turn on/off lights
- open/close curtain


## Building

Download dependencies:
```sh
$ npm install
```

Build production (minified):
```sh
$ npm run build
```

Start dev server on <http://localhost:8080/> and automatically build on save:
```sh
$ npm start
```
